591     /**
592      * Waits for the files matched by the specified {@code pathPattern} to be changed since the specified
593      * {@code lastKnownRevision}. If no changes were made within 1 minute, the returned
594      * {@link CompletableFuture} will be completed with {@code null}.
595      *
596      * @return the latest known {@link Revision} which contains the changes for the matched files.
597      *         {@code null} if the files were not changed for 1 minute since the invocation of this method.
